# Sistema Administrativo

Este projeto contém um backend em Node.js (Express) e um frontend em React (Vite) para um sistema administrativo.

## Estrutura
- `backend/`: API REST para autenticação, usuários e recursos administrativos
- `frontend/`: Interface web para clientes

## Como iniciar

### Backend
1. Instale as dependências:
   ```
   cd backend
   npm install
   ```
2. Inicie o servidor:
   ```
   npm run dev
   ```

### Frontend
1. Instale as dependências:
   ```
   cd frontend
   npm install
   ```
2. Inicie o servidor de desenvolvimento:
   ```
   npm run dev
   ```

Acesse o frontend em `http://localhost:5173` e o backend em `http://localhost:3001`.
